## Getting Started

### Prerequisites
- macOS 10.15 or later
- Flutter SDK
- Android device with USB debugging enabled

### Installation
1. Clone the repository
```bash
git clone https://github.com/sam-baraka/flutter_android_file_transfer.git
```

2. Install dependencies
```bash
flutter pub get
```

3. Run the app
```bash
flutter run
```
